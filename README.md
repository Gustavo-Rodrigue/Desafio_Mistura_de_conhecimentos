# 📚 Desafio Mistura de Conhecimentos

Este repositório contém um desafio que une todas os assuntos cementados no tecnólogo de Ciência de dados até o momento atual do curso.

Caso queira ler um pouco sobre o Briefing do desafio acesse o PDF [Desafio](https://github.com/Gustavo-Rodrigue/Desafio_Mistura_de_conhecimentos/blob/main/Desafio.pdf)

## 🚀 Objetivo

Este projeto tem como objeto testar os conhecimentos e como juntá-los em um único projeto, assim simulando o processo real de uma empresa de alto rendimento.

Manter um aprendizado contínuo (**Lifelong Learning**) em áreas estratégicas, unindo competências técnicas (**Hard Skills**) com uma visão crítica.
